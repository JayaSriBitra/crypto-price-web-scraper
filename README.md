# Crypto Price Web Scraper

This is a Python project that scrapes cryptocurrency prices from a website and stores the data in a CSV file.

## Technologies Used

* Python
* Requests
* BeautifulSoup
* Pandas
* Jupyter Notebook

## Project Files

CryptoPricesWebScraper.ipynb → Main scraping code
crypto_prices.csv → Output dataset

## How to Run

1. Install required libraries:

pip install -r requirements.txt

2. Open the notebook:

CryptoPricesWebScraper.ipynb

3. Run all cells to scrape crypto prices.
